Kuis Jaringan Komputer - Application Layer

Bab 3 - Protokol dan Layanan Layer Aplikasi
Total Soal 25
Bab 3
Waktu 60 Menit
1
Application Layer dalam model OSI dan TCP/IP berfungsi sebagai...
a. Penghubung antara aplikasi yang digunakan dengan jaringan di bawahnya
b. Pengatur aliran data antar perangkat
c. Pengelola koneksi fisik jaringan
d. Penghubung antara router dan switch
Jawaban: a. Penghubung antara aplikasi yang digunakan dengan jaringan di bawahnya
Penjelasan: Application Layer berfungsi sebagai antarmuka antara aplikasi yang digunakan pengguna dengan protokol jaringan di layer bawahnya, memungkinkan aplikasi untuk mengakses layanan jaringan.
2
Dua konsep penting dalam Application Layer adalah...
a. Transport Layer dan Network Layer
b. Application Layer dan Application Software
c. Client dan Server
d. Protocol dan Service
Jawaban: b. Application Layer dan Application Software
Penjelasan: Dua konsep penting adalah Application Layer (lapisan protokol) dan Application Software (program aplikasi yang menggunakan layanan jaringan).
3
Fungsi utama dari Presentation Layer adalah...
a. Membuat dan menjaga dialog antar aplikasi
b. Pengkodean, konversi, kompresi, dan enkripsi data
c. Pengalamatan perangkat dalam jaringan
d. Pengaturan koneksi fisik
Jawaban: b. Pengkodean, konversi, kompresi, dan enkripsi data
Penjelasan: Presentation Layer bertanggung jawab untuk representasi data, termasuk konversi format, enkripsi, dekripsi, dan kompresi data.
4
Protocol yang berfungsi untuk menerjemahkan nama domain menjadi alamat IP adalah...
a. HTTP
b. DNS
c. SMTP
d. FTP
Jawaban: b. DNS
Penjelasan: DNS (Domain Name System) adalah protokol yang menerjemahkan nama domain yang mudah diingat (seperti google.com) menjadi alamat IP numerik (seperti 172.217.0.142).
5
Protocol yang digunakan untuk transfer file yang membentuk halaman web adalah...
a. FTP
b. SMTP
c. HTTP
d. Telnet
Jawaban: c. HTTP
Penjelasan: HTTP (Hypertext Transfer Protocol) digunakan untuk mentransfer halaman web dan konten terkait dari web server ke web browser.
6
Protocol yang digunakan untuk transfer mail messages dan attachments adalah...
a. POP3
b. IMAP
c. SMTP
d. HTTP
Jawaban: c. SMTP
Penjelasan: SMTP (Simple Mail Transfer Protocol) digunakan untuk mengirim email dan attachment antar server email.
7
Protocol yang digunakan untuk memberikan alamat IP secara dinamis ke host adalah...
a. DNS
b. DHCP
c. HTTP
d. FTP
Jawaban: b. DHCP
Penjelasan: DHCP (Dynamic Host Configuration Protocol) secara otomatis memberikan alamat IP dan konfigurasi jaringan lainnya ke perangkat yang terhubung ke jaringan.
8
Dalam model client/server, keuntungan utama adalah...
a. Desentralisasi resources
b. Sentralisasi resources
c. Tidak perlu maintenance
d. Biaya implementasi rendah
Jawaban: b. Sentralisasi resources
Penjelasan: Keuntungan utama model client/server adalah sentralisasi sumber daya dan data, yang memudahkan manajemen, backup, dan keamanan.
9
Server daemon berfungsi untuk...
a. Mengontrol akses user
b. "Mendengarkan" request dari client
c. Mengenkripsi data
d. Mengkompresi file
Jawaban: b. "Mendengarkan" request dari client
Penjelasan: Server daemon adalah program yang berjalan di background dan terus-menerus "mendengarkan" permintaan dari client pada port tertentu.
10
Port number yang digunakan oleh Telnet adalah...
a. Port 21
b. Port 23
c. Port 25
d. Port 80
Jawaban: b. Port 23
Penjelasan: Telnet menggunakan port 23 untuk koneksi remote ke perangkat jaringan. Port 21 untuk FTP, port 25 untuk SMTP, dan port 80 untuk HTTP.
11
Dalam sistem DNS, program yang berjalan pada client disebut...
a. Named
b. Resolver
c. Daemon
d. Server
Jawaban: b. Resolver
Penjelasan: DNS resolver adalah program pada client yang bertanggung jawab untuk mengirimkan query DNS dan menerima respons dari server DNS.
12
Utility yang digunakan untuk melihat cache DNS pada Windows adalah...
a. nslookup
b. ipconfig /displaydns
c. ipconfig /flushdns
d. ping
Jawaban: b. ipconfig /displaydns
Penjelasan: Perintah "ipconfig /displaydns" digunakan untuk menampilkan isi cache DNS resolver pada sistem Windows.
13
Protocol yang digunakan untuk komunikasi web yang aman adalah...
a. HTTP
b. HTTPS
c. FTP
d. Telnet
Jawaban: b. HTTPS
Penjelasan: HTTPS (HTTP Secure) adalah versi aman dari HTTP yang menggunakan enkripsi SSL/TLS untuk melindungi data yang ditransmisikan.
14
Protocol yang digunakan untuk download email dan menghapusnya dari server adalah...
a. SMTP
b. IMAP
c. POP3
d. HTTP
Jawaban: c. POP3
Penjelasan: POP3 (Post Office Protocol version 3) biasanya mendownload email dari server ke client dan menghapusnya dari server setelah didownload.
15
FTP menggunakan dua port, yaitu...
a. Port 20 dan 21
b. Port 21 dan 22
c. Port 23 dan 24
d. Port 25 dan 26
Jawaban: a. Port 20 dan 21
Penjelasan: FTP menggunakan port 21 untuk koneksi kontrol (perintah) dan port 20 untuk koneksi data (transfer file).
16
Proses DHCP yang pertama adalah...
a. DHCP Offer
b. DHCP Request
c. DHCP Discover
d. DHCP Acknowledgement
Jawaban: c. DHCP Discover
Penjelasan: Proses DHCP dimulai dengan DHCP Discover, dimana client mengirimkan broadcast untuk mencari server DHCP yang tersedia.
17
Protocol yang digunakan untuk file sharing dalam jaringan Microsoft adalah...
a. FTP
b. SMB
c. HTTP
d. DNS
Jawaban: b. SMB
Penjelasan: SMB (Server Message Block) adalah protokol yang digunakan untuk berbagi file, printer, dan resource lainnya dalam jaringan Microsoft Windows.
18
Kelemahan utama dari jaringan Peer-to-Peer adalah...
a. Sentralisasi resources
b. Desentralisasi services
c. Biaya implementasi tinggi
d. Keamanan yang terlalu ketat
Jawaban: b. Desentralisasi services
Penjelasan: Kelemahan utama P2P adalah desentralisasi layanan, yang menyulitkan manajemen, backup, dan keamanan data karena tidak ada kontrol terpusat.
19
Protocol yang lebih aman dibandingkan Telnet adalah...
a. HTTP
b. FTP
c. SSH
d. SMTP
Jawaban: c. SSH
Penjelasan: SSH (Secure Shell) adalah protokol yang lebih aman dibandingkan Telnet karena menggunakan enkripsi untuk melindungi data yang ditransmisikan.
20
Dalam email system, program yang menerima email dari MTA dan menempatkannya ke mailbox user disebut...
a. MUA
b. MTA
c. MDA
d. DNS
Jawaban: c. MDA
Penjelasan: MDA (Mail Delivery Agent) bertanggung jawab untuk menerima email dari MTA dan menempatkannya ke dalam mailbox pengguna yang sesuai.
21
Jenis record DNS yang digunakan untuk mail exchange adalah...
a. A record
b. NS record
c. CNAME record
d. MX record
Jawaban: d. MX record
Penjelasan: MX (Mail Exchange) record digunakan untuk menentukan server email yang bertanggung jawab menerima email untuk suatu domain.
22
Protocol yang memungkinkan user mengakses email tanpa mendownloadnya adalah...
a. POP3
b. IMAP
c. SMTP
d. FTP
Jawaban: b. IMAP
Penjelasan: IMAP (Internet Message Access Protocol) memungkinkan pengguna mengakses dan mengelola email langsung di server tanpa perlu mendownloadnya ke perangkat lokal.
23
Web server yang populer termasuk kecuali...
a. Apache
b. Microsoft IIS
c. DNS Server
d. Nginx
Jawaban: c. DNS Server
Penjelasan: DNS Server bukan web server, melainkan server yang menangani permintaan DNS. Apache, IIS, dan Nginx adalah contoh web server populer.
24
Protocol yang digunakan dalam P2P file sharing adalah...
a. SMB
b. Gnutella
c. HTTP
d. FTP
Jawaban: b. Gnutella
Penjelasan: Gnutella adalah protokol P2P yang digunakan untuk berbagi file secara terdesentralisasi, populer digunakan oleh aplikasi seperti LimeWire.
25
Fungsi utama Session Layer adalah...
a. Enkripsi data
b. Membuat dan menjaga dialog antar aplikasi
c. Konversi format data
d. Kompresi data
Jawaban: b. Membuat dan menjaga dialog antar aplikasi
Penjelasan: Session Layer bertanggung jawab untuk membangun, mengelola, dan mengakhiri sesi komunikasi antara aplikasi pada perangkat yang berbeda.
