Bab 6 - Pengalamatan IP dan Subnetting

Bab 6

1
IPv4 address terdiri dari...
a. 16 bit
b. 32 bit
c. 64 bit
d. 128 bit
Jawaban: b. 32 bit
Penjelasan: IPv4 address terdiri dari 32 bit yang dibagi menjadi 4 oktet (8 bit per oktet), menghasilkan total sekitar 4.3 miliar alamat unik.
2
Format penulisan IPv4 address yang umum digunakan adalah...
a. Binary murni
b. Dotted decimal notation
c. Hexadecimal
d. Scientific notation
Jawaban: b. Dotted decimal notation
Penjelasan: Dotted decimal notation adalah format penulisan IPv4 yang paling umum, dimana 32 bit dibagi menjadi 4 oktet yang dipisahkan oleh titik (contoh: 192.168.1.1).
3
Bagian dari IP address yang identik untuk semua host dalam network yang sama disebut...
a. Host portion
b. Network portion
c. Subnet mask
d. Broadcast address
Jawaban: b. Network portion
Penjelasan: Network portion adalah bagian dari alamat IP yang sama untuk semua perangkat dalam jaringan yang sama, sedangkan host portion unik untuk setiap perangkat.
4
Konversi binary 11000011 ke decimal adalah...
a. 195
b. 193
c. 192
d. 194
Jawaban: a. 195
Penjelasan: 11000011 dalam binary = (1×128) + (1×64) + (0×32) + (0×16) + (0×8) + (0×4) + (1×2) + (1×1) = 128 + 64 + 2 + 1 = 195.
5
Konversi decimal 172 ke binary adalah...
a. 10101100
b. 10101101
c. 10101110
d. 10101111
Jawaban: a. 10101100
Penjelasan: 172 dalam decimal = 128 + 32 + 8 + 4 = 172, sehingga binary-nya adalah 10101100.
6
Tiga jenis address dalam IPv4 network range adalah...
a. Network, Host, dan Gateway address
b. Network, Broadcast, dan Host address
c. Public, Private, dan Special address
d. Unicast, Multicast, dan Broadcast address
Jawaban: b. Network, Broadcast, dan Host address
Penjelasan: Dalam setiap subnet IPv4, terdapat tiga jenis alamat: Network address (identifikasi jaringan), Broadcast address (untuk semua host), dan Host addresses (untuk perangkat individual).
7
Address yang digunakan untuk mengirim packet ke semua host dalam network adalah...
a. Network address
b. Broadcast address
c. Host address
d. Default gateway
Jawaban: b. Broadcast address
Penjelasan: Broadcast address digunakan untuk mengirim paket ke semua host dalam jaringan tertentu. Contoh: 192.168.1.255 untuk network 192.168.1.0/24.
8
Address 127.0.0.1 digunakan untuk...
a. Default gateway
b. Loopback testing
c. Public Internet
d. Network broadcast
Jawaban: b. Loopback testing
Penjelasan: Alamat 127.0.0.1 adalah loopback address yang digunakan untuk menguji konfigurasi TCP/IP pada komputer lokal tanpa mengirim paket ke jaringan.
9
Range private address untuk Class B adalah...
a. 10.0.0.0 - 10.255.255.255
b. 172.16.0.0 - 172.31.255.255
c. 192.168.0.0 - 192.168.255.255
d. 169.254.0.0 - 169.254.255.255
Jawaban: b. 172.16.0.0 - 172.31.255.255
Penjelasan: Range private address untuk Class B adalah 172.16.0.0 hingga 172.31.255.255, yang mencakup 16 blok jaringan Class B.
10
Organisasi global yang bertanggung jawab untuk assignment IPv4 address adalah...
a. ISP
b. IANA
c. IEEE
d. ISO
Jawaban: b. IANA
Penjelasan: IANA (Internet Assigned Numbers Authority) adalah organisasi global yang bertanggung jawab untuk mengkoordinasi assignment alamat IP dan parameter protokol Internet lainnya.
11
Metode assignment address dimana administrator mengkonfigurasi manual setiap device disebut...
a. Dynamic addressing
b. Static addressing
c. DHCP
d. Automatic addressing
Jawaban: b. Static addressing
Penjelasan: Static addressing adalah metode dimana administrator secara manual mengkonfigurasi alamat IP pada setiap perangkat, berbeda dengan dynamic addressing yang menggunakan DHCP.
12
Notasi /24 dalam IPv4 address setara dengan subnet mask...
a. 255.0.0.0
b. 255.255.0.0
c. 255.255.255.0
d. 255.255.255.255
Jawaban: c. 255.255.255.0
Penjelasan: Notasi /24 berarti 24 bit untuk network portion, yang setara dengan subnet mask 255.255.255.0 dalam dotted decimal notation.
13
Proses yang digunakan device network untuk menentukan network portion dari IP address disebut...
a. ORing
b. ANDing
c. NOTing
d. XORing
Jawaban: b. ANDing
Penjelasan: Proses ANDing logical antara alamat IP dan subnet mask digunakan untuk menentukan network portion dari alamat IP.
14
Untuk network 192.168.1.0/24, broadcast address-nya adalah...
a. 192.168.1.0
b. 192.168.1.1
c. 192.168.1.254
d. 192.168.1.255
Jawaban: d. 192.168.1.255
Penjelasan: Untuk network 192.168.1.0/24, broadcast address adalah 192.168.1.255, dimana semua bit host portion di-set ke 1.
15
Jumlah usable host addresses untuk network /26 adalah...
a. 62
b. 64
c. 126
d. 128
Jawaban: a. 62
Penjelasan: Untuk /26, terdapat 6 bit host (32-26=6), sehingga total host = 2^6 = 64. Setelah dikurangi network dan broadcast address, usable host = 64 - 2 = 62.
16
Tujuan utama subnetting adalah...
a. Mengurangi keamanan network
b. Membuat network lebih besar
c. Membagi network menjadi smaller subnets
d. Menghilangkan need untuk routers
Jawaban: c. Membagi network menjadi smaller subnets
Penjelasan: Tujuan utama subnetting adalah membagi jaringan besar menjadi subnetwork yang lebih kecil untuk meningkatkan efisiensi, keamanan, dan manajemen.
17
Dalam subnetting, bits yang dipinjam dari...
a. Network portion
b. Host portion
c. Subnet mask
d. Broadcast address
Jawaban: b. Host portion
Penjelasan: Dalam subnetting, bit dipinjam dari host portion untuk membuat subnet ID tambahan, sehingga mengurangi jumlah host per subnet tetapi meningkatkan jumlah subnet.
18
Magic number dalam subnetting digunakan untuk menentukan...
a. Jumlah routers yang dibutuhkan
b. Size setiap subnet
c. Kecepatan network
d. Jumlah ISP yang diperlukan
Jawaban: b. Size setiap subnet
Penjelasan: Magic number dalam subnetting adalah nilai dari bit terkecil dalam subnet mask yang digunakan untuk menentukan ukuran blok setiap subnet.
19
Untuk network 172.16.0.0/16 yang di-subnet menjadi /24, jumlah subnets yang dihasilkan adalah...
a. 16
b. 64
c. 256
d. 512
Jawaban: c. 256
Penjelasan: Dari /16 ke /24, terdapat 8 bit tambahan untuk subnetting (24-16=8), sehingga jumlah subnet = 2^8 = 256.
20
Protocol yang digunakan oleh perintah ping adalah...
a. TCP
b. UDP
c. ICMP
d. HTTP
Jawaban: c. ICMP
Penjelasan: Perintah ping menggunakan ICMP (Internet Control Message Protocol) Echo Request dan Echo Reply untuk menguji konektivitas jaringan.
21
Perintah yang digunakan untuk melihat path yang dilalui packet ke destination adalah...
a. ping
b. ipconfig
c. tracert/traceroute
d. nslookup
Jawaban: c. tracert/traceroute
Penjelasan: Tracert (Windows) atau traceroute (Linux/Unix) digunakan untuk melacak rute yang dilalui paket dari sumber ke tujuan dan mengidentifikasi setiap hop.
22
Default route direpresentasikan sebagai...
a. 127.0.0.1
b. 0.0.0.0
c. 255.255.255.255
d. 192.168.0.1
Jawaban: b. 0.0.0.0
Penjelasan: Default route direpresentasikan sebagai 0.0.0.0/0, yang berarti "semua jaringan" dan digunakan ketika tidak ada route yang lebih spesifik.
23
Address 169.254.x.x secara otomatis diassign ketika...
a. Device terkoneksi ke Internet
b. DHCP server tidak tersedia
c. Network menggunakan static addressing
d. Device berada di network public
Jawaban: b. DHCP server tidak tersedia
Penjelasan: Alamat 169.254.x.x adalah APIPA (Automatic Private IP Addressing) yang secara otomatis diassign ketika perangkat tidak dapat mendapatkan alamat dari DHCP server.
24
Dalam classful addressing, address 130.61.22.204 termasuk Class...
a. A
b. B
c. C
d. D
Jawaban: b. B
Penjelasan: Alamat 130.61.22.204 termasuk Class B karena oktet pertamanya (130) berada dalam range 128-191. Class B memiliki 16 bit network dan 16 bit host.
25
Keuntungan menggunakan DHCP untuk address assignment adalah...
a. Konfigurasi yang lebih kompleks
b. Administrative overhead yang lebih rendah
c. Keamanan yang lebih tinggi
d. Performa yang lebih cepat
Jawaban: b. Administrative overhead yang lebih rendah
Penjelasan: Keuntungan utama DHCP adalah mengurangi administrative overhead karena alamat IP dikonfigurasi secara otomatis, menghilangkan kebutuhan untuk konfigurasi manual pada setiap perangkat.
