Kuis Jaringan Komputer - Transport Layer

Bab 4 - Protokol TCP dan UDP
Total Soal 25
Bab 4
Waktu 60 Menit
1
Transport Layer dalam model OSI berfungsi untuk...
a. Menghubungkan aplikasi dengan jaringan fisik
b. Menyediakan layanan transportasi data dari host ke tujuan
c. Mengatur pengalamatan IP pada jaringan
d. Mengelola koneksi antar router
Jawaban: b. Menyediakan layanan transportasi data dari host ke tujuan
Penjelasan: Transport Layer bertanggung jawab untuk menyediakan layanan end-to-end yang andal untuk mentransfer data antara host sumber dan host tujuan.
2
Dua protokol utama yang bekerja pada Transport Layer adalah...
a. HTTP dan FTP
b. TCP dan UDP
c. IP dan ICMP
d. DNS dan DHCP
Jawaban: b. TCP dan UDP
Penjelasan: TCP (Transmission Control Protocol) dan UDP (User Datagram Protocol) adalah dua protokol utama yang beroperasi pada Transport Layer.
3
Karakteristik utama dari TCP adalah...
a. Connectionless dan unreliable
b. Connection-oriented dan reliable
c. Low overhead dan best effort
d. Tidak ada flow control
Jawaban: b. Connection-oriented dan reliable
Penjelasan: TCP adalah protokol yang connection-oriented (membutuhkan pembentukan koneksi) dan reliable (andal) dengan mekanisme acknowledgments dan retransmission.
4
Karakteristik utama dari UDP adalah...
a. Connection-oriented dan reliable
b. Connectionless dan best effort delivery
c. Error checking dan flow control
d. Sequencing dan acknowledgements
Jawaban: b. Connectionless dan best effort delivery
Penjelasan: UDP adalah protokol yang connectionless (tidak membutuhkan pembentukan koneksi) dan menggunakan best effort delivery (pengiriman sebaik mungkin tanpa jaminan).
5
Fungsi utama segmentation pada Transport Layer adalah...
a. Mengenkripsi data sebelum dikirim
b. Membagi data menjadi bagian-bagian yang lebih kecil
c. Mengkompresi data untuk menghemat bandwidth
d. Menyembunyikan alamat IP
Jawaban: b. Membagi data menjadi bagian-bagian yang lebih kecil
Penjelasan: Segmentation adalah proses membagi data aplikasi menjadi segmen-segmen yang lebih kecil untuk memudahkan transmisi dan meningkatkan efisiensi jaringan.
6
Port number yang termasuk dalam well-known ports range adalah...
a. 0 - 1023
b. 1024 - 49151
c. 49152 - 65535
d. 65536 - 70000
Jawaban: a. 0 - 1023
Penjelasan: Well-known ports berkisar dari 0 hingga 1023 dan digunakan untuk layanan-layanan standar seperti HTTP (port 80), FTP (port 21), dan SSH (port 22).
7
Port number 80 biasanya digunakan untuk protokol...
a. FTP
b. HTTP
c. SMTP
d. DNS
Jawaban: b. HTTP
Penjelasan: Port 80 adalah port standar untuk HTTP (Hypertext Transfer Protocol) yang digunakan untuk komunikasi web.
8
Port number 53 digunakan oleh kedua protokol TCP dan UDP untuk layanan...
a. HTTP
b. DNS
c. FTP
d. SMTP
Jawaban: b. DNS
Penjelasan: Port 53 digunakan oleh DNS (Domain Name System) untuk kedua protokol TCP (untuk zone transfer) dan UDP (untuk query biasa).
9
Proses "three-way handshake" dilakukan oleh protokol...
a. UDP
b. TCP
c. HTTP
d. DNS
Jawaban: b. TCP
Penjelasan: Three-way handshake adalah proses yang digunakan oleh TCP untuk membentuk koneksi yang andal antara dua host sebelum pertukaran data dimulai.
10
Urutan yang benar dalam three-way handshake adalah...
a. SYN, SYN-ACK, ACK
b. ACK, SYN, SYN-ACK
c. SYN, ACK, FIN
d. SYN, SYN, ACK
Jawaban: a. SYN, SYN-ACK, ACK
Penjelasan: Urutan three-way handshake: 1) Client mengirim SYN, 2) Server merespons dengan SYN-ACK, 3) Client mengirim ACK untuk menyelesaikan pembentukan koneksi.
11
Field dalam header TCP yang digunakan untuk mengontrol flow data disebut...
a. Sequence Number
b. Acknowledgement Number
c. Window Size
d. Checksum
Jawaban: c. Window Size
Penjelasan: Window Size field dalam header TCP digunakan untuk flow control, menunjukkan jumlah data yang dapat diterima oleh penerima sebelum memerlukan acknowledgment.
12
Mekanisme yang digunakan TCP untuk mengirim ulang segment yang hilang disebut...
a. Flow Control
b. Error Recovery
c. Retransmission
d. Segmentation
Jawaban: c. Retransmission
Penjelasan: Retransmission adalah mekanisme TCP untuk mengirim ulang segmen yang hilang atau rusak berdasarkan timeout atau duplicate acknowledgments.
13
Protokol yang cocok untuk aplikasi real-time seperti VoIP dan online games adalah...
a. TCP
b. UDP
c. HTTP
d. FTP
Jawaban: b. UDP
Penjelasan: UDP cocok untuk aplikasi real-time seperti VoIP dan game online karena overhead yang rendah dan tidak ada delay akibat retransmission.
14
Kombinasi antara IP address dan port number disebut...
a. Segment
b. Datagram
c. Socket
d. Frame
Jawaban: c. Socket
Penjelasan: Socket adalah kombinasi unik dari alamat IP dan port number yang mengidentifikasi endpoint komunikasi tertentu dalam jaringan.
15
Contoh aplikasi yang menggunakan TCP adalah...
a. DNS
b. DHCP
c. HTTP
d. TFTP
Jawaban: c. HTTP
Penjelasan: HTTP menggunakan TCP karena membutuhkan koneksi yang andal untuk mentransfer halaman web dan konten terkait.
16
Contoh aplikasi yang menggunakan UDP adalah...
a. FTP
b. Telnet
c. SMTP
d. DNS
Jawaban: d. DNS
Penjelasan: DNS biasanya menggunakan UDP untuk query biasa karena overhead yang rendah dan kecepatan respons yang lebih cepat.
17
Fungsi dari sequence numbers dalam TCP adalah...
a. Mengidentifikasi aplikasi
b. Melacak urutan segment untuk reassembly
c. Menentukan kecepatan transfer
d. Mengenkripsi data
Jawaban: b. Melacak urutan segment untuk reassembly
Penjelasan: Sequence numbers digunakan untuk melacak urutan segmen TCP sehingga penerima dapat menyusun ulang data dalam urutan yang benar.
18
Dalam TCP, flow control dilakukan dengan mengatur...
a. Sequence numbers
b. Window size
c. Port numbers
d. MAC addresses
Jawaban: b. Window size
Penjelasan: Flow control dalam TCP dilakukan dengan mengatur window size, yang menentukan jumlah data yang dapat dikirim sebelum memerlukan acknowledgment.
19
Protokol yang tidak melakukan proses connection establishment adalah...
a. TCP
b. UDP
c. HTTP
d. FTP
Jawaban: b. UDP
Penjelasan: UDP adalah protokol connectionless yang tidak melakukan proses connection establishment seperti three-way handshake pada TCP.
20
Port number yang digunakan untuk HTTPS adalah...
a. 80
b. 443
c. 21
d. 25
Jawaban: b. 443
Penjelasan: Port 443 adalah port standar untuk HTTPS (HTTP Secure) yang menggunakan enkripsi SSL/TLS untuk komunikasi web yang aman.
21
Proses pengiriman data dimana pengirim menunggu acknowledgement sebelum mengirim data berikutnya disebut...
a. Full-duplex
b. Stop-and-wait
c. Three-way handshake
d. Flow control
Jawaban: b. Stop-and-wait
Penjelasan: Stop-and-wait adalah metode sederhana dimana pengirim mengirim satu paket dan menunggu acknowledgment sebelum mengirim paket berikutnya.
22
Keuntungan utama UDP dibandingkan TCP adalah...
a. Reliabilitas yang tinggi
b. Overhead yang rendah
c. Flow control yang baik
d. Error recovery yang handal
Jawaban: b. Overhead yang rendah
Penjelasan: Keuntungan utama UDP adalah overhead yang rendah karena tidak ada mekanisme connection establishment, flow control, atau retransmission.
23
Dalam TCP, field yang menandakan tidak ada data lagi yang akan dikirim adalah...
a. SYN
b. ACK
c. FIN
d. RST
Jawaban: c. FIN
Penjelasan: FIN (Finish) flag dalam header TCP menandakan bahwa pengirim telah selesai mengirim data dan ingin menutup koneksi.
24
Protokol yang menggunakan two-way handshake untuk termination adalah...
a. UDP
b. TCP
c. HTTP
d. DNS
Jawaban: b. TCP
Penjelasan: TCP menggunakan two-way handshake untuk termination koneksi, dimana setiap sisi mengirim FIN dan menerima ACK.
25
Aplikasi yang membutuhkan keandalan tinggi dalam pengiriman data sebaiknya menggunakan...
a. UDP
b. TCP
c. DNS
d. DHCP
Jawaban: b. TCP
Penjelasan: Aplikasi yang membutuhkan keandalan tinggi seperti transfer file, email, dan web browsing sebaiknya menggunakan TCP karena mekanisme reliability-nya.
