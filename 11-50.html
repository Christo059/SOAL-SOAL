Kuis Jaringan Komputer - 50 Soal

11. Apa fungsi utama dari router dalam jaringan?
a. Menghubungkan jaringan berbeda dan menentukan jalur terbaik
b. Menghubungkan perangkat dalam jaringan lokal saja
c. Menyimpan data sementara
d. Memperkuat sinyal kabel
Jawaban: a. Menghubungkan jaringan berbeda dan menentukan jalur terbaik
Penjelasan: Router menghubungkan beberapa jaringan serta menentukan jalur terbaik agar paket sampai ke tujuan.
12. Apa jenis topologi jaringan yang semua perangkat terhubung ke satu titik pusat?
a. Bus
b. Star
c. Ring
d. Mesh
Jawaban: b. Star
Penjelasan: Dalam topologi star, semua perangkat terhubung ke perangkat pusat seperti switch atau hub.
13. Apa kelemahan utama topologi bus?
a. Mahal biaya instalasi
b. Jika satu perangkat rusak, seluruh jaringan bermasalah
c. Sulit menambah perangkat baru
d. Menggunakan banyak kabel
Jawaban: b. Jika satu perangkat rusak, seluruh jaringan bermasalah
Penjelasan: Pada topologi bus, gangguan kabel utama menyebabkan seluruh jaringan tidak berfungsi.
14. Pada model TCP/IP, layer apa yang bertanggung jawab untuk pengepakan data menjadi paket?
a. Application
b. Transport
c. Internet
d. Network Access
Jawaban: b. Transport
Penjelasan: Layer transport menangani segmentasi data dan pembentukan segmen untuk pengiriman.
15. Apa singkatan LAN?
a. Large Area Network
b. Local Area Network
c. Long Area Network
d. Local Access Network
Jawaban: b. Local Area Network
Penjelasan: LAN adalah jaringan yang mencakup area fisik terbatas seperti rumah atau kantor.
16. Apa fungsi utama dari switch dalam jaringan?
a. Menghubungkan jaringan berbeda
b. Merutekan paket ke alamat MAC yang tepat
c. Memperkuat sinyal jaringan
d. Membagi jaringan menjadi segmen yang lebih kecil
Jawaban: b. Merutekan paket ke alamat MAC yang tepat
Penjelasan: Switch bekerja di layer data link dengan memfilter dan meneruskan frame ke perangkat tujuan sesuai alamat MAC.
17. Apa yang dimaksud dengan bandwidth dalam jaringan komputer?
a. Jumlah perangkat dalam jaringan
b. Jumlah data maksimum yang dapat dikirim dalam satu waktu
c. Kecepatan jaringan secara keseluruhan
d. Jenis kabel yang dipakai
Jawaban: b. Jumlah data maksimum yang dapat dikirim dalam satu waktu
Penjelasan: Bandwidth merujuk pada kapasitas maksimum transmisi data melalui media jaringan.
18. Manakah media transmisi yang menggunakan gelombang cahaya untuk mengirim data?
a. Kabel tembaga
b. Fiber Optik
c. Gelombang radio
d. Ethernet
Jawaban: b. Fiber Optik
Penjelasan: Fiber optik menggunakan pulsa cahaya untuk mengirimkan data dengan kecepatan tinggi dan jarak jauh.
19. Apa fungsi dari protokol ICMP?
a. Mengirim email
b. Mengirim pesan error dan status jaringan
c. Memindahkan file antar komputer
d. Mempercepat pengiriman data
Jawaban: b. Mengirim pesan error dan status jaringan
Penjelasan: ICMP digunakan untuk mengirim pesan kontrol dan kesalahan jaringan antar host dan router.
20. Fungsi DHCP di jaringan adalah...
a. Memberikan alamat IP otomatis ke perangkat
b. Mengatur rute paket data
c. Melindungi jaringan dari serangan
d. Mengubah data menjadi sinyal
Jawaban: a. Memberikan alamat IP otomatis ke perangkat
Penjelasan: DHCP berfungsi untuk secara otomatis memberikan alamat IP dan konfigurasi jaringan ke perangkat baru.
21. Apa yang dimaksud dengan protokol HTTP?
a. Protokol pengiriman email
b. Protokol untuk mengakses halaman web
c. Protokol pengiriman file
d. Protokol pengelolaan alamat IP
Jawaban: b. Protokol untuk mengakses halaman web
Penjelasan: HTTP adalah protokol standar untuk mentransfer dokumen web dari server ke browser.
22. Apa fungsi utama DNS dalam jaringan?
a. Mengatur alamat IP perangkat
b. Menerjemahkan nama domain ke alamat IP
c. Mengelola protokol TCP dan UDP
d. Mengirim data secara aman
Jawaban: b. Menerjemahkan nama domain ke alamat IP
Penjelasan: DNS berfungsi mengubah nama yang mudah diingat menjadi alamat IP yang digunakan perangkat jaringan.
23. Perangkat manakah yang digunakan untuk menghubungkan jaringan lokal ke jaringan luas (WAN)?
a. Hub
b. Switch
c. Router
d. Access Point
Jawaban: c. Router
Penjelasan: Router memfasilitasi koneksi antara jaringan lokal dengan jaringan lain yang lebih luas.
24. Apa yang dimaksud dengan protokol TCP?
a. Protokol pengiriman data tanpa koneksi
b. Protokol komunikasi yang andal dan berorientasi koneksi
c. Protokol pengiriman suara saja
d. Protokol pengaturan alamat IP
Jawaban: b. Protokol komunikasi yang andal dan berorientasi koneksi
Penjelasan: TCP menyediakan layanan transfer data dengan pengecekan kesalahan dan urutan data yang tepat.
25. Jenis kabel yang paling umum digunakan di jaringan LAN adalah...
a. Kabel Coaxial
b. Kabel UTP
c. Kabel Fiber Optik
d. Kabel HDMI
Jawaban: b. Kabel UTP
Penjelasan: Kabel UTP (Unshielded Twisted Pair) sangat umum digunakan karena biaya dan kemudahan instalasi.
26. Apa fungsi subnetting dalam jaringan IP?
a. Menggabungkan beberapa jaringan menjadi satu
b. Membagi jaringan IP menjadi beberapa subnet lebih kecil
c. Menghubungkan satu komputer dengan internet
d. Mengganti alamat IP secara otomatis
Jawaban: b. Membagi jaringan IP menjadi beberapa subnet lebih kecil
Penjelasan: Subnetting membantu mengatur jaringan untuk meningkatkan efisiensi dan keamanan.
27. Protokol mana yang menggunakan nomor port 80?
a. FTP
b. HTTP
c. SMTP
d. DNS
Jawaban: b. HTTP
Penjelasan: Port 80 adalah port standar untuk protokol HTTP pada web browsing.
28. Apa itu VPN?
a. Virtual Public Network
b. Virtual Private Network
c. Virtual Public Node
d. Virtual Private Node
Jawaban: b. Virtual Private Network
Penjelasan: VPN adalah jaringan privat yang menggunakan jaringan publik untuk menghubungkan perangkat secara aman.
29. Apa yang dimaksud latency dalam jaringan?
a. Kapasitas transmisi data
b. Waktu delay saat data dikirimkan dari sumber ke tujuan
c. Jumlah data yang dikirim dalam satu detik
d. Kecepatan pengkodean data
Jawaban: b. Waktu delay saat data dikirimkan dari sumber ke tujuan
Penjelasan: Latency adalah waktu yang diperlukan sinyal untuk berpindah dari sumber ke tujuan.
30. Peran firewall dalam jaringan adalah?
a. Mengoptimalkan kecepatan jaringan
b. Mengontrol dan melindungi akses ke jaringan
c. Menyimpan data sementara
d. Menghubungkan jaringan berbeda
Jawaban: b. Mengontrol dan melindungi akses ke jaringan
Penjelasan: Firewall adalah sistem keamanan yang mengatur lalu lintas data untuk mencegah akses tidak sah.
31. Apa bedanya protokol TCP dan UDP?
a. TCP lebih cepat dari UDP
b. UDP tidak menjamin pengiriman data yang andal, TCP menjamin
c. TCP hanya untuk email, UDP untuk web
d. TCP versi lama, UDP versi baru
Jawaban: b. UDP tidak menjamin pengiriman data yang andal, TCP menjamin
Penjelasan: TCP menggunakan koneksi berorientasi dan pemeriksaan error, sedangkan UDP lebih ringan tapi tidak menjamin pengiriman.
32. Apa contoh perangkat jaringan yang bekerja di layer 2 OSI?
a. Router
b. Switch
c. Hub
d. Firewall
Jawaban: b. Switch
Penjelasan: Switch beroperasi di layer 2 (Data Link) OSI untuk meneruskan frame berdasarkan MAC address.
33. Apa tujuan utama dari protokol ARP?
a. Mengubah IP menjadi MAC address
b. Mengirim email
c. Menghubungkan router dengan switch
d. Mengatur alamat IP
Jawaban: a. Mengubah IP menjadi MAC address
Penjelasan: ARP (Address Resolution Protocol) menerjemahkan alamat IP ke alamat fisik MAC.
34. Apa yang dimaksud dengan bandwidth delay product?
a. Produk dari kecepatan dan delay jaringan
b. Alat untuk meningkatkan bandwidth
c. Jumlah data yang bisa dikirim tanpa error
d. Ukuran buffer router
Jawaban: a. Produk dari kecepatan dan delay jaringan
Penjelasan: Ini menunjukkan jumlah data dalam transit yang jaringan dapat tangani sesuai kapasitas dan delay.
35. Apa yang dimaksud dengan jitter dalam jaringan?
a. Variasi delay antar paket data
b. Jumlah paket yang hilang
c. Kesalahan pengiriman data
d. Kapasitas jaringan
Jawaban: a. Variasi delay antar paket data
Penjelasan: Jitter adalah variasi waktu keterlambatan pengiriman paket yang bisa mempengaruhi kualitas real-time.
36. Apa fungsi NAT dalam jaringan?
a. Mengubah alamat IP lokal menjadi alamat IP publik
b. Menghubungkan perangkat ke internet
c. Menyimpan data jaringan
d. Memblokir virus
Jawaban: a. Mengubah alamat IP lokal menjadi alamat IP publik
Penjelasan: NAT (Network Address Translation) memungkinkan banyak perangkat menggunakan satu alamat IP publik.
37. Apa jenis media yang paling tahan terhadap interferensi elektromagnetik?
a. Kabel coaxial
b. Kabel Fiber Optik
c. Kabel UTP
d. Gelombang radio
Jawaban: b. Kabel Fiber Optik
Penjelasan: Fiber optik menggunakan cahaya sehingga tidak terpengaruh oleh interferensi elektromagnetik.
38. Apa yang disebut dengan protokol SMTP?
a. Protokol transfer file
b. Protokol pengiriman email
c. Protokol pengambilan email
d. Protokol browsing web
Jawaban: b. Protokol pengiriman email
Penjelasan: SMTP (Simple Mail Transfer Protocol) digunakan untuk mengirim email antar server.
39. Apa fungsi switch dalam jaringan?
a. Memperluas sinyal jaringan
b. Meneruskan data ke perangkat berdasarkan alamat MAC
c. Menghubungkan jaringan berbeda
d. Menyimpan data sementara
Jawaban: b. Meneruskan data ke perangkat berdasarkan alamat MAC
Penjelasan: Switch membaca frame dan meneruskannya hanya ke perangkat yang sesuai berdasarkan MAC address.
40. Apa nama protokol yang digunakan untuk mengambil email dari server?
a. SMTP
b. POP3
c. FTP
d. HTTP
Jawaban: b. POP3
Penjelasan: POP3 (Post Office Protocol version 3) digunakan untuk mengunduh email dari server ke klien.
41. Apa manfaat utama VLAN dalam jaringan?
a. Menghubungkan jaringan secara fisik
b. Membagi jaringan menjadi segmen logis yang terpisah
c. Menyederhanakan alamat IP
d. Meningkatkan kecepatan kabel
Jawaban: b. Membagi jaringan menjadi segmen logis yang terpisah
Penjelasan: VLAN memungkinkan pemisahan lalu lintas jaringan secara logis tanpa memandang lokasi fisik.
42. Apa nama protokol yang digunakan untuk resolusi nama host menjadi alamat IP?
a. DHCP
b. DNS
c. ARP
d. ICMP
Jawaban: b. DNS
Penjelasan: DNS menerjemahkan nama domain yang mudah diingat oleh manusia menjadi alamat IP komputer.
43. Apa yang dilakukan oleh perangkat modem dalam jaringan?
a. Menghubungkan jaringan lokal ke internet
b. Mengubah sinyal digital ke sinyal analog dan sebaliknya
c. Memperkuat sinyal kabel
d. Mengatur alamat IP
Jawaban: b. Mengubah sinyal digital ke sinyal analog dan sebaliknya
Penjelasan: Modem mengubah data digital komputer menjadi sinyal analog untuk transmisi melalui saluran telepon.
44. Apa fungsi dari protokol FTP?
a. Mengirim email
b. Transfer file antar komputer di jaringan
c. Mengatur alamat IP
d. Menyimpan data di server
Jawaban: b. Transfer file antar komputer di jaringan
Penjelasan: FTP (File Transfer Protocol) memungkinkan transfer file di antara komputer melalui jaringan.
45. Apa yang dimaksud dengan topologi mesh?
a. Semua perangkat terhubung langsung ke perangkat pusat
b. Semua perangkat terhubung satu sama lain secara langsung
c. Semua perangkat terhubung dalam satu kabel utama
d. Semua perangkat terhubung secara acak
Jawaban: b. Semua perangkat terhubung satu sama lain secara langsung
Penjelasan: Topologi mesh menyediakan koneksi point-to-point antar semua perangkat di jaringan.
46. Apa tujuan dari protokol Telnet?
a. Mengakses perangkat jaringan dari jarak jauh
b. Mengirim email
c. Transfer file
d. Konfigurasi IP
Jawaban: a. Mengakses perangkat jaringan dari jarak jauh
Penjelasan: Telnet memberikan akses terminal ke perangkat lain melalui jaringan.
47. Apa perbedaan antara LAN dan WAN?
a. LAN mencakup area kecil, WAN mencakup area besar
b. LAN menggunakan kabel, WAN tidak
c. WAN lebih cepat dari LAN
d. LAN untuk internet, WAN untuk lokal
Jawaban: a. LAN mencakup area kecil, WAN mencakup area besar
Penjelasan: LAN adalah jaringan lokal di area terbatas, WAN menghubungkan jaringan yang tersebar luas.
48. Apa fungsi protokol SNMP?
a. Mengatur alamat IP
b. Monitoring dan pengelolaan perangkat jaringan
c. Mentranfer file
d. Mengirim email
Jawaban: b. Monitoring dan pengelolaan perangkat jaringan
Penjelasan: SNMP (Simple Network Management Protocol) digunakan untuk mengelola dan memantau perangkat jaringan.
49. Apa yang dimaksud dengan firewall stateful?
a. Firewall yang memeriksa status koneksi untuk menentukan apakah paket diperbolehkan
b. Firewall yang hanya mengizinkan trafik masuk
c. Firewall yang menyimpan data jaringan
d. Firewall yang menghubungkan jaringan
Jawaban: a. Firewall yang memeriksa status koneksi untuk menentukan apakah paket diperbolehkan
Penjelasan: Stateful firewall melacak keadaan koneksi untuk pengamanan yang lebih canggih.
50. Apa fungsi dari protokol ICMP?
a. Mengatur pengiriman email
b. Mengirim pesan status dan error antar perangkat jaringan
c. Membantu transfer file
d. Merutekan paket data
Jawaban: b. Mengirim pesan status dan error antar perangkat jaringan
Penjelasan: ICMP memberikan informasi kontrol dan kesalahan dalam jaringan IP.
