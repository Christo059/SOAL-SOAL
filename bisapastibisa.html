Kuis Jaringan Komputer - Network Layer

Bab 5 - IP Addressing dan Routing
Total Soal 25
Bab 5
Waktu 60 Menit
1
Protokol utama yang digunakan pada Network Layer adalah...
a. TCP
b. UDP
c. IPv4
d. HTTP
Jawaban: c. IPv4
Penjelasan: IPv4 (Internet Protocol version 4) adalah protokol utama yang digunakan pada Network Layer untuk pengalamatan logis dan routing paket data.
2
Karakteristik utama dari IPv4 adalah...
a. Connection-oriented dan reliable
b. Connectionless dan best effort delivery
c. Hanya bekerja pada media tertentu
d. Selalu membutuhkan acknowledgement
Jawaban: b. Connectionless dan best effort delivery
Penjelasan: IPv4 adalah protokol connectionless (tanpa koneksi) yang menggunakan best effort delivery (pengiriman sebaik mungkin) tanpa jaminan paket sampai.
3
MTU (Maximum Transmission Unit) berkaitan dengan...
a. Kecepatan maksimum transfer data
b. Ukuran maksimum packet yang dapat dikirim melalui media
c. Jumlah maksimum router dalam jaringan
d. Waktu maksimum untuk packet delivery
Jawaban: b. Ukuran maksimum packet yang dapat dikirim melalui media
Penjelasan: MTU (Maximum Transmission Unit) adalah ukuran maksimum paket data yang dapat ditransmisikan melalui media jaringan tertentu tanpa perlu fragmentasi.
4
Proses memecah packet menjadi bagian yang lebih kecil ketika melewati media dengan MTU lebih kecil disebut...
a. Segmentation
b. Encapsulation
c. Fragmentation
d. Reassembly
Jawaban: c. Fragmentation
Penjelasan: Fragmentation adalah proses memecah paket IP menjadi fragmen-fragmen yang lebih kecil ketika paket harus melewati jaringan dengan MTU yang lebih kecil.
5
Alasan utama untuk membagi jaringan besar menjadi subnetworks adalah KECUALI...
a. Meningkatkan performa jaringan
b. Meningkatkan keamanan
c. Memudahkan manajemen alamat
d. Menambah jumlah broadcast domain
Jawaban: d. Menambah jumlah broadcast domain
Penjelasan: Subnetting justru mengurangi ukuran broadcast domain, bukan menambahnya. Tujuan subnetting adalah meningkatkan performa, keamanan, dan memudahkan manajemen.
6
Bagian dari IP address yang identik untuk semua host dalam network yang sama adalah...
a. Host portion
b. Network portion
c. Subnet mask
d. Default gateway
Jawaban: b. Network portion
Penjelasan: Network portion adalah bagian dari alamat IP yang identik untuk semua host dalam jaringan yang sama, sedangkan host portion unik untuk setiap host.
7
Notasi 192.168.1.0/24 menunjukkan bahwa...
a. 24 bit pertama adalah host portion
b. 24 bit pertama adalah network portion
c. Total 24 bit untuk seluruh address
d. 24 bit terakhir adalah network portion
Jawaban: b. 24 bit pertama adalah network portion
Penjelasan: Notasi /24 menunjukkan bahwa 24 bit pertama dari alamat IP adalah network portion, dan 8 bit sisanya adalah host portion.
8
Fungsi utama router dalam jaringan adalah...
a. Memperkuat sinyal data
b. Menghubungkan host dalam network yang sama
c. Mem-forward packet antar network yang berbeda
d. Mengenkripsi data yang dikirim
Jawaban: c. Mem-forward packet antar network yang berbeda
Penjelasan: Fungsi utama router adalah meneruskan paket data antara jaringan yang berbeda berdasarkan informasi alamat IP tujuan.
9
Dua address yang dimiliki setiap host adalah...
a. IP address dan port number
b. MAC address dan IP address
c. Hostname dan domain name
d. Subnet mask dan default gateway
Jawaban: b. MAC address dan IP address
Penjelasan: Setiap host memiliki MAC address (alamat fisik layer 2) dan IP address (alamat logis layer 3) untuk komunikasi jaringan.
10
Default gateway biasanya adalah...
a. Alamat IP dari switch
b. Alamat IP dari router interface di network tersebut
c. Alamat IP dari server DNS
d. Alamat IP dari host sendiri
Jawaban: b. Alamat IP dari router interface di network tersebut
Penjelasan: Default gateway adalah alamat IP dari antarmuka router yang terhubung ke jaringan lokal, digunakan untuk mengirim paket ke jaringan lain.
11
Selama packet forwarding, alamat yang TIDAK berubah adalah...
a. Source MAC address
b. Destination MAC address
c. Source IP address
d. Semua alamat berubah
Jawaban: c. Source IP address
Penjelasan: Source IP address tetap tidak berubah selama packet forwarding dari sumber ke tujuan, sedangkan MAC address berubah di setiap hop.
12
Routing table berisi informasi tentang...
a. Hanya network yang directly connected
b. Hanya network remote
c. Both directly connected dan remote networks
d. Hanya default route
Jawaban: c. Both directly connected dan remote networks
Penjelasan: Routing table berisi informasi tentang jaringan yang terhubung langsung (directly connected) dan jaringan remote yang dipelajari melalui routing protocol.
13
Ketika router tidak menemukan route spesifik untuk destination packet, maka akan menggunakan...
a. Broadcast route
b. Default route
c. Local route
d. Static route
Jawaban: b. Default route
Penjelasan: Default route (biasanya 0.0.0.0/0) digunakan ketika router tidak menemukan route yang lebih spesifik untuk tujuan paket.
14
Static routing berarti...
a. Route berubah secara otomatis
b. Route dikonfigurasi secara manual
c. Route learned dari routing protocol
d. Route hanya untuk network lokal
Jawaban: b. Route dikonfigurasi secara manual
Penjelasan: Static routing adalah metode dimana administrator jaringan mengkonfigurasi route secara manual ke routing table router.
15
Contoh routing protocol adalah...
a. HTTP
b. OSPF
c. TCP
d. UDP
Jawaban: b. OSPF
Penjelasan: OSPF (Open Shortest Path First) adalah contoh routing protocol yang digunakan untuk mempelajari dan berbagi informasi routing secara dinamis.
16
Keuntungan dynamic routing dibanding static routing adalah...
a. Konfigurasi yang lebih sederhana
b. Tidak memerlukan pengetahuan network topology
c. Otomatis beradaptasi dengan perubahan topology
d. Lebih aman karena tidak ada informasi yang di-share
Jawaban: c. Otomatis beradaptasi dengan perubahan topology
Penjelasan: Keuntungan utama dynamic routing adalah kemampuan untuk secara otomatis beradaptasi dengan perubahan topologi jaringan tanpa intervensi manual.
17
Proses dimana router menentukan path terbaik untuk mengirim packet disebut...
a. Packet forwarding
b. Path determination
c. Address resolution
d. Encapsulation
Jawaban: b. Path determination
Penjelasan: Path determination adalah proses dimana router menganalisis routing table untuk menentukan path terbaik menuju tujuan paket.
18
Pada interface serial, router...
a. Selalu membutuhkan MAC address destination
b. Tidak membutuhkan MAC address untuk forwarding
c. Hanya menggunakan IP address source
d. Menggunakan broadcast address
Jawaban: b. Tidak membutuhkan MAC address untuk forwarding
Penjelasan: Pada interface serial (point-to-point), router tidak membutuhkan MAC address karena koneksi langsung antara dua perangkat tanpa multiple hosts.
19
Router mem-block...
a. Semua packet dari network berbeda
b. Broadcast packets secara default
c. Packet dengan IP address invalid
d. Semua packet kecuali yang ke default gateway
Jawaban: b. Broadcast packets secara default
Penjelasan: Router secara default memblokir paket broadcast untuk mencegah broadcast storm dan membatasi broadcast domain.
20
Hierarchical addressing memungkinkan router untuk...
a. Mengetahui alamat setiap host di internet
b. Hanya perlu mengetahui network portion untuk routing decision
c. Menggunakan MAC address untuk routing
d. Membuat routing table yang sangat besar
Jawaban: b. Hanya perlu mengetahui network portion untuk routing decision
Penjelasan: Hierarchical addressing memungkinkan router membuat keputusan routing berdasarkan network portion saja, tanpa perlu mengetahui setiap host individual.
21
Subnet mask 255.255.255.0 sama dengan...
a. /16
b. /24
c. /32
d. /8
Jawaban: b. /24
Penjelasan: Subnet mask 255.255.255.0 sama dengan /24 dalam notasi CIDR, yang berarti 24 bit untuk network portion dan 8 bit untuk host portion.
22
Alamat yang digunakan untuk komunikasi dalam network yang sama adalah...
a. Default gateway
b. Broadcast address
c. MAC address
d. Public IP address
Jawaban: c. MAC address
Penjelasan: Untuk komunikasi dalam jaringan yang sama (LAN), perangkat menggunakan MAC address untuk mengidentifikasi satu sama lain di layer 2.
23
Ketika host mengirim packet ke network yang berbeda, packet akan dikirim ke...
a. Switch
b. Default gateway
c. DNS server
d. Host destination langsung
Jawaban: b. Default gateway
Penjelasan: Ketika host ingin mengirim paket ke jaringan yang berbeda, paket akan dikirim ke default gateway (router) untuk diteruskan ke jaringan tujuan.
24
Routing table pada host biasanya berisi...
a. Semua route ke seluruh network
b. Hanya route ke network langsung terhubung
c. Host address dan default gateway
d. Hanya default route
Jawaban: c. Host address dan default gateway
Penjelasan: Routing table pada host biasanya sederhana, berisi route ke jaringan lokal dan default gateway untuk semua traffic ke jaringan lain.
25
Proses query untuk mendapatkan MAC address dari IP address disebut...
a. DNS lookup
b. ARP request
c. Routing protocol
d. TCP handshake
Jawaban: b. ARP request
Penjelasan: ARP (Address Resolution Protocol) request digunakan untuk menemukan MAC address yang terkait dengan IP address tertentu dalam jaringan lokal.
